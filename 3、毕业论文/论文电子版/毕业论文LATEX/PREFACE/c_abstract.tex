
\chapter*{摘\quad 要}

%\setcounter{page}{1}

%\vspace{0.5cm}
心脏的电生理模拟就是通过计算机模拟跨膜电势在心肌细胞中的传播规律，它是研究心脏的生理和病理特征的重要手段。然而心肌细胞具有非均匀性和复杂的导电率特性，因此原整数阶~FHN 模型模拟跨膜电势具有很大的局限性。考虑到分数阶导数的非局部特性及其在细胞离子反常扩散领域的应用，非线性分数阶~FHN 模型更能有效地描述心肌细胞的记忆性和遗传性等重要性质。由于很难导出非线性分数阶~FHN 模型的解析解，于是探索分数阶~FHN 模型的数值解法成为研究的必然趋势。本文建立了求解非线性分数阶~FHN 模型的标准$\backslash$非标准有限差分方法，并证明了所提方法的稳定性与收敛性。

首先，本文介绍了分数阶导数的基本理论，引入了标准~Gr\"{u}mwald-Letnikov 分数阶导数定义的~Riesz 空间分数阶~FHN 模型。为了获得求解~Riesz 空间分数阶~FHN 模型的稳定差分离散格式，引进了移位~Gr\"{u}mwald-Letnikov 分数阶导数定义，实现了刘发旺等人提出的无条件稳定性的标准有限差分格式。

其次，考虑非标准有限差分格式的非局部特性，本文提出了基于~Taylor 展开的非标准有限差分法，有效求解了移位~Gr\"{u}mwald-Letnikov 分数阶导数定义的~Riesz 空间分数阶~FHN 模型，并证明了该格式的稳定性和收敛性：当~$2r\beta < {H_2}$ 时，取~$\tau  < 1/2({H_2} - 2r\beta )$，该非标准有限差分格式稳定且收敛。

最后，本文通过两个数值算例来验证方法的稳定性与收敛性：针对二维~Riesz 空间分数阶反应-扩散方程，给出了标准有限差分格式和非标准有限差分格式的数值误差、时间与空间收敛阶；针对二维~Riesz 空间分数阶~FHN 模型，在不同分数阶导数及扩散系数下，成功模拟了跨膜电势所形成的周期性螺旋波。在数值计算过程中，采用了多重网格方法，有效提高了数值格式的计算效率。
